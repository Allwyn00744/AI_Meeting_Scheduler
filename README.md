# AI Meeting Scheduler

An AI-powered Meeting Scheduler and Calendar Assistant that automates meeting scheduling, calendar management, and AI-driven meeting assistance.

## Features

- Smart Meeting Scheduling
- AI Scheduling Assistant
- Google Calendar Integration
- Microsoft Outlook Integration
- Voice & Text Scheduling
- Meeting Summaries
- Action Item Generation
- Conflict Detection
- Time Zone Management
- Notifications

## Getting Started

### Clone Repository

```bash
git clone https://github.com/Allwyn00744/AI_Meeting_Scheduler.git
```

### Navigate to Backend

```bash
cd AI_Meeting_Scheduler/backend
```

### Create Virtual Environment

```bash
python -m venv .venv
```

### Activate Virtual Environment

Windows

```bash
.venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run the Application

```bash
python run.py
```

Server:

```
http://127.0.0.1:8000
```
